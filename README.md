# PC 성능 오버레이 (PC Performance Overlay)

Windows 환경을 위한 초경량, 고가독성 실시간 PC 리소스 모니터링 오버레이 프로그램입니다.

![PCOverlay](https://github.com/h2y-hyhong2/h2y-system-overlay/raw/main/tray-icon.png) *상태 표시줄 트레이 아이콘을 통해 다양한 제어를 지원합니다.*

---

## 🌟 주요 특징 (Features)

1. **초미니멀 & 고선명 텍스트 디자인**
   - 불필요한 차트, 그래프 및 윈도우 프레임을 완전히 제거하여 화면 공간을 절약합니다.
   - 글자 외곽에 **4방향 블랙 테두리 아웃라인**을 추가하여 어떤 배경 위에서도 선명하게 읽힙니다.
   - 전체 폰트 스케일을 **1.5배(30px)** 확대 조정하여 시인성이 매우 우수합니다.

2. **초경량 모니터링 엔진 (0%대 CPU 점유율)**
   - 외부 WMIC 프로세스 쿼리를 제거하고, Node.js 네이티브 하드웨어 바인딩(`os` 모듈)을 이용해 CPU 및 RAM 사용률을 조회하여 백그라운드 리소스 소모를 극도로 최소화했습니다.

3. **10단계 계단식 레드 경고 필터**
   - CPU/RAM 리소스 소모량에 따라 10% 단위(10단계)로 색상이 계단식으로 변화합니다.
   - 부하 수준이 90% 임계점을 초과하면 강렬한 빨간색(Red)으로 고정되어 위협 상황을 즉시 시각적으로 알립니다.

4. **실시간 가로/세로 레이아웃 변환**
   - **가로 모드 (기본값, 500x30)**: 단일 행 구조로 화면 상/하단 구석에 콤팩트하게 상주시키기 적합합니다.
   - **세로 모드 (160x150)**: CPU, RAM, Network 데이터를 세로로 나열하여 정렬해 줍니다.
   - **더블 클릭 전환**: 오버레이 창 아무 곳이나 마우스로 **더블 클릭**하면 가로 ⇄ 세로 모드가 실시간으로 즉시 스위칭됩니다.

5. **마우스 통과 및 위치 고정 제어**
   - 트레이 메뉴에서 **'마우스 클릭 통과'**를 켜면 오버레이가 보인 채로 아래 창들을 정상 클릭할 수 있습니다.
   - **'위치 고정 (드래그 금지)'**을 켜면 실수로 마우스 드래그를 하여 오버레이 위치가 이탈하는 것을 차단합니다.

## 🚀 로컬 실행 방법 (How to Run Locally)

본 프로그램을 로컬 개발 환경에서 직접 실행하려면 [Node.js](https://nodejs.org/)가 사전 설치되어 있어야 합니다.

1. **의존성 모듈 설치**
   ```bash
   npm install
   ```

2. **개발 모드로 프로그램 실행**
   ```bash
   npm run dev
   ```
   *(오버레이 창 활성화 상태에서 `Ctrl + R`을 누르면 실시간 코드 변경 내역이 즉각 리로드됩니다.)*

---

## 📦 포터블 EXE 파일 빌드 방법 (How to Build EXE)

다른 PC에서 설치 과정 없이 즉시 단독 실행할 수 있는 무설치 포터블 단일 파일(`.exe`)을 컴파일하는 방법입니다.

1. **배포용 EXE 컴파일 실행**
   ```bash
   npm run build
   ```
2. **산출물 확인**
   - 빌드가 완료되면 루트 폴더 하위의 `dist/` 디렉토리에 **`PCOverlay.exe`** 파일이 생성됩니다.
   - 해당 파일을 복사하여 모니터링이 필요한 어느 윈도우 PC에서든 자유롭게 실행하시면 됩니다.
   *※ 빌드 실행 전에 구버전 오버레이 프로그램이 켜져 있다면, 프로세스를 완전히 종료(`트레이 아이콘 우클릭 -> 종료`)한 상태에서 빌드를 가동하셔야 파일 락 에러를 방지할 수 있습니다.*

## ⚙️ 설정 가이드 (Settings Guide)

- **오버레이 위치 이동**: 오버레이 영역을 마우스 왼쪽 버튼으로 잡고 원하는 위치로 클릭 드래그하여 임의 배치합니다.
- **항상 위에 노출**: 윈도우 화면 가장 상위에 떠 있도록 유지시킵니다. (트레이 메뉴에서 On/Off 토글)
- **종료 및 닫기**: 마우스 호버 시 우측 상단에 노출되는 종료 버튼(`×`)을 누르거나, 트레이 아이콘 우클릭 컨텍스트 메뉴에서 `종료`를 누르면 프로세스가 안전하게 닫힙니다.
